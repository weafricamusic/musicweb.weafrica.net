🔔 Push Notification Smoke Test

Complete step-by-step guide for testing push notifications

1 Login & Collect Tokens

  • Login to app with Firebase Auth
  • Get Firebase ID Token
  • Get FCM Device Token
  • Collect device info
final helper = PushSmokeTestHelper(); final tokens = await helper.collectTokens();
✅ Firebase UID, FCM Token, Device Model

2 Register Device Token

  • Send token to backend
  • Backend saves to Supabase
  • Verify 200 OK response
await helper.registerDeviceToken( idToken: tokens['id_token'], fcmToken: tokens['fcm_token'], platform: tokens['platform'], deviceModel: tokens['device_model'], );
✅ 200 OK with token ID

3 Verify in Supabase

  • Open Supabase SQL Editor
  • Run verification query
  • Confirm token exists
  • Check is_active = true
SELECT * FROM notification_device_tokens WHERE fcm_token = 'YOUR_TOKEN'
✅ One row found with correct user_id

4 Send Test Notification

  • Open Admin Dashboard
  • Go to Notifications > Push
  • Select topic (system/consumers)
  • Click "Send Now"
Config: - Topic: system - Country: gh - Title: Test Message
✅ Status: Sent to 1 device

5 Receive on Device

  • Watch for notification alert
  • Check console for onMessage
  • Tap notification
  • Verify correct screen opens

6 Test Rate-Limiting

  • Send notification twice
  • First: expect 200 OK
  • Second: expect 429 Limited
final test = PushRateLimitTest(); await test.runRateLimitTest();
✅ First OK, Second Limited (429)
